## 🔍 Project Overview
An end-to-end **Data Analytics & Visualization project** using **Power BI** to analyze **job market trends, unemployment patterns, and skill demand** from 2019–2023.

This dashboard helps:
- Identify **unemployment hotspots**
- Analyze **in-demand skills**
- Detect **skill gaps**
- Recommend **targeted retraining strategies**

---

## 🚀 Key Features
- 📈 Interactive dashboards (3 pages)
- 📊 KPI tracking (Unemployment, Job Postings, MoM Change)
- 🧠 Custom **Retraining Priority Index**
- 📍 Region-wise comparison
- 🛠 Advanced DAX calculations
- 🎯 Insight cards for decision-making
- 🎨 Professional UI with Microsoft Fabric themes

---

## 🛠 Tech Stack
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)**
- **Power Query**
- **Data Modeling (Star Schema)**

## ⚙️ Key Metrics

### 📌 Retraining Priority Index
A composite score based on:
- Unemployment Rate
- Skill Gap Index
- Trend (Hotspot Score)

➡️ Helps prioritize **regions needing workforce intervention**

---

## 📈 Key Insights
- Regional disparities in unemployment trends
- High job demand ≠ low unemployment (skill mismatch)
- Concentration of demand in select technical skills
- Dynamic changes in retraining needs over time

---

## 🎯 Business Impact
- Supports **data-driven policymaking**
- Helps identify **skill gaps**
- Assists workforce planning & training strategy
- Enables **targeted upskilling programs**

---

## ⚠️ Assumptions & Limitations
- Skill demand inferred from job postings (proxy-based)
- Job postings distributed proportionally across skills
- Retraining Priority is a relative index
